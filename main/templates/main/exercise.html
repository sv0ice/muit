Git Tutorial

Git - это система контроля версий.

Git помогает вам отслеживать изменения кода.

Git используется для совместной работы над кодом.

Изучение по примерам

В этом руководстве мы покажем вам команды Git, подобные этой:

Example

$ git --version

git version 2.30.2.windows.1

Для новых пользователей использование представления терминала может показаться немного сложным. Не волнуйся! Мы постараемся сделать это действительно простым, и изучение этого способа даст вам хорошее представление о том, как работает Git.

В приведенном выше коде вы можете увидеть команды (ввод) и вывод.

Строки, подобные этой, - это команды, которые мы вводим:

Example

$ git --version

Строки, подобные этой, являются выводом / ответом на наши команды:

Example

git version 2.30.2.windows.1

В общем, вводятся строки с $ перед ними. Это команды, которые вы можете скопировать и запустить в своем терминале.
