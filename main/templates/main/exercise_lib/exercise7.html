Git Help

Если у вас возникли проблемы с запоминанием команд или параметров для команд, вы можете воспользоваться справкой Git.

Есть несколько различных способов, которыми вы можете использовать команду help в командной строке:

  • git command -help - Посмотреть все доступные параметры для конкретной команды
  • git help --all - Посмотреть все возможные команды

Подробнее можно узнать в официальной документации
