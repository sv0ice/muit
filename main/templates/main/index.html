Введение в Git

Git — это распределенная система контроля версий, широко используемая для контроля версий и совместной работы над проектами.

Basic Git Commands

  • git init - инициализирует гит репозиторий.
  • git clone - клонирует репозиторий на локальную машину
  • git add - добавляет файлы в промежуточную область..
  • git commit - сохраняет изменения в репозитории.
  • git push - отправляет изменения на удаленный сервер.
  • git pull - извлекает последние изменения с удаленного сервера.

Working with Branches

Git позволяет создавать ветки и управлять ими, упрощая параллельную разработку и интеграцию изменений.
