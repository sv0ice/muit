Обучение Git

Введение в Git

Git — это распределенная система контроля версий, широко используемая для контроля версий и совместной работы над проектами.

Basic Git Commands

  • git init - creates a new Git repository.
  • git clone - copies a repository to the local machine.
  • git add - adds files to the staging area.
  • git commit - saves changes in the repository.
  • git push - sends changes to a remote server.
  • git pull - retrieves the latest changes from a remote server.

Working with Branches

Git allows creating and managing branches, making parallel development and change integration easier.
